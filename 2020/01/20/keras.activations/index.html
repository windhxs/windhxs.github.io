本文总结介绍tf.keras中的一些激活函数

tf.keras.activations.elu

elu函数表达式如下
$$
f(x) = \begin{cases}
x & \text{ if } x>=0 \\
\alpha*(e^x-1) & \text{ if } x<0
\end{cases}
$$
超参数只有$\alpha$,查看tf.keras中关于elu的代码如下

参数：

  • x 输入的张量
  • alpha 一个常数

返回值:elu激活函数

tf.keras.activations.exponential

exponential函数表达式如下
$$
f(x) = e^x
$$
没有超参数，查看tf.keras中关于exponential的代码如下

参数

  • x 输入的张量

返回值

  • exp激活函数

tf.keras.activations.linear

linear函数表达式如下
$$
f(x) = x
$$
没有超参数，查看tf.keras中关于exponential的代码如下

参数

  • x 输入的张量

返回值

tf.keras.activations.rule

rule函数表达式如下
$$
f(x) = max(0,x)
$$
,而leaky rule表达式为
$$
f(x) = max(0.1x,x)
$$
在tf.keras中，对于rule有了更细致的定义，查看关于rule的代码如下

翻译一下就是
$$
f(x) = \begin{cases}
max\underline{}value & \text{if} x >= max\underline{}value\\
x & \text{ if } x < max\underline{}alue\ &\ x>=threshold \\
\alpha*x & \text{ if } x <max\underline{}value\ &\ x <threshold
\end{cases}
$$
默认情况下就是普通的rule函数，即f(x)=max(0,x)

参数

  • x 输入的张量
  • alpha 左半部分的斜率
  • max_value 整个函数能产生的最大值
  • threshold 两个部分的分界线

返回值

  • rule 激活函数

tf.keras.activations.selu

selu函数表达式如下
$$
f(x) = \begin{cases}
scale*x & \text{ if } x>=0 \\
scale *\alpha * (e^x -1)& \text{ if } x<0
\end{cases}
$$
其中$scale$和$\alpha$已经是确定的

  • $scale$ : 1.05070098
  • $\alpha$ : 1.67326324
    tf.keras中关于selu的代码如下 参数
  • x 输入的张量

返回值

  • selu激活函数

selu的目的主要是调整$x>=0$时函数的斜率

sigmoid函数表达式如下

tf.keras.activations.softmax

softmax函数表达式如下
$$
f_i(x) = \frac{e^{x_i}}{\sum_{j}e^{x_j}}
$$

参数

  • x 输入的张量

返回值

  • 张量
