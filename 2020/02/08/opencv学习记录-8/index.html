opencv学习记录#8

Posted on 2020-02-08 Edited on 2020-02-11

图像卷积

参数

  • src 原图像
  • ddpth 所需深度，-1即可
  • kernel 卷积核，矩阵
  • dst 生成图像
  • anchor 内核的锚点
  • delta 卷积后要加上的偏置
  • borderType 边缘填充类型
    • cv.BORDER_REPLICATE 填充最外层的像素值
    • cv.BORDER_CONSTANT 填充常数
    • cv.BORDER_REFLECT 翻转
    • cv.BORDER_WRAP 平移
    • cv.BORDER_REFLECT_101 除了最边上的像素翻转
    • cv.BORDER_TRANSPARENT

返回值

dst生成的图像

高斯滤波

用高斯核卷积

参数

  • src 原图像
  • ksize 卷积核大小
  • sigmaX 卷积后x方向的标准差
  • dst 生成图像
  • sigmaY 卷积核y方向的标准差
  • anchor 卷积核核的锚点
  • borderType 边缘填充类型
    • cv.BORDER_REPLICATE 填充最外层的像素值
    • cv.BORDER_CONSTANT 填充常数
    • cv.BORDER_REFLECT 翻转
    • cv.BORDER_WRAP 平移
    • cv.BORDER_REFLECT_101 除了最边上的像素翻转
    • cv.BORDER_TRANSPARENT

返回值

dst生成的图像

双边滤波

双边滤波可以很好的保留边缘的信息

参数

  • src 原图像
  • d 卷积核半径，建议5
  • sigmaColor 颜色空间的标准差
  • sigmaSpace 距离空间的标准差
  • dst 生成图像
  • borderType 边缘填充类型
    • cv.BORDER_REPLICATE 填充最外层的像素值
    • cv.BORDER_CONSTANT 填充常数
