opencv学习记录#4

Posted on 2020-02-04 Edited on 2020-02-09

在图片中画图形

画一条线段

参数

  • img 原图片
  • pt1、pt2 线段的两个端点
  • color 线段的颜色，三个元素的元组(b,g,r)
  • thickness 线段的宽度
  • lineType 线段类型
    • cv.LINE_4 4联通的线
    • cv.LINE_8 8联通的线
    • cv.LINE_AA 抗锯齿的线

返回值

画过线之后的图片

  • lineType 线段类型

画过矩形之后的图片

画椭圆

参数

  • img 原图片
  • center 椭圆中心
  • axes 两个轴的一半，二维元组
  • angle 椭圆旋转角度
  • startAngle 椭圆弧的起始角度
  • endAngle 椭圆弧的终止角度
  • color 线段的颜色，三个元素的元组(b,g,r)
  • thickness 线段的宽度
  • lineType 线段类型
    • cv.LINE_4 4联通的线
    • cv.LINE_8 8联通的线
    • cv.LINE_AA 抗锯齿的线

返回值

画过椭圆之后的图片

画多边形

参数

  • img 原图片
  • pts 顶点集合
  • isClosed 该多边形是否闭合，即最后一个点和第一个点是否连起来
  • color 线段的颜色，三个元素的元组(b,g,r)
  • thickness 线段的宽度
  • lineType 线段类型
    • cv.LINE_4 4联通的线
    • cv.LINE_8 8联通的线
    • cv.LINE_AA 抗锯齿的线

返回值

画过多边形之后的图片

添加文字

参数

  • img 原图片
  • text 要显示的内容
  • org 距离图片左下角的位置
  • fontFace 字体
    • cv.FONT_HERSHEY_SIMPLEX 正常sans-serif字体
    • cv.FONT_HERSHEY_PLAIN 小号sans-serif字体
    • cv.FONT_HERSHEY_DUPLEX 正常sans-serif字体(更复杂)
    • cv.FONT_HERSHEY_COMPLEX 正常serif字体
    • cv.FONT_HERSHEY_TRIPLEX 正常serif字体(更复杂)
    • cv.FONT_HERSHEY_SCRIPT_SIMPLEX 手写体
    • cv.FONT_HERSHEY_SCRIPT_COMPLEX 手写体(更复杂)
    • cv.FONT_ITALIC 斜体
  • fontScale 字体放缩
  • color 字体的颜色，三个元素的元组(b,g,r)
  • thickness 线段的宽度
  • lineType 线段类型
    • cv.LINE_4 4联通的线
    • cv.LINE_8 8联通的线
    • cv.LINE_AA 抗锯齿的线
  • bottomLeftOrigin 文字是否在左下角

返回值

添加完文字后的图片
