ImageDataGenerator类

Posted on 2020-01-31

本文自我总结ImageDataGenerator类

ImageDataGenerator通常用于创造图像数据，并且对数据进行预处理

__init__方法

源码

参数

  • featurewise_center 是否将整个数据集的均值变为0
  • samplewise_center 是否将每张图片均值变为0
  • featurewise_std_normalization 是否除以整个数据集的标准差，即标准化
  • samplewise_std_normalization 是否除以自身的标准差
  • zca_whitening 是否进行zca白化
  • zca_epsilon zca白化过程中为了避免出现除0而分母需要增加的一个很小的数
  • rotation_range 图片随机转动的角度的范围，即图片在0到该范围内随机旋转
  • width_shift_range 图片水平偏移的幅度
  • height_shift_range 图片垂直偏移的幅度
  • shear_range 错切变换
  • zoom_range 随机缩放的幅度
  • channel_shift_range 随机通道偏移的幅度
  • fill_mode ‘constant’，‘nearest’，‘reflect’或‘wrap’,当变换后的图片产生了空档该如何填充
  • cval fill_mode=constant时的填充值
  • horizontal_flip 进行随机水平翻转
  • vertical_flip 进行随机竖直翻转
  • rescale 放缩因子，将该因子乘上数据，该操作会在所有的变换之前进行
  • preprocessing_function 被应用于每个输入的函数，最后进行，该函数必须接受一张图片，并返回相同形状的np.array
  • data_format ‘channels_first’或者‘channels_last’
  • validation_split 验证数据集所占比例
  • dtype 生成的数据类型

返回值

ImageDataGenerator类对象

apply_transform方法

参数

  • x 三维张量，单张图片
  • transform_parameters 字典，主要参数有
    • 'theta': Float. Rotation angle in degrees.
    • 'tx': Float. Shift in the x direction.
    • 'ty': Float. Shift in the y direction.
    • 'shear': Float. Shear angle in degrees.
    • 'zx': Float. Zoom in the x direction.
    • 'zy': Float. Zoom in the y direction.
    • 'flip_horizontal': Boolean. Horizontal flip.
    • 'flip_vertical': Boolean. Vertical flip.
    • 'channel_shift_intencity': Float. Channel shift intensity.
    • 'brightness': Float. Brightness shift intensity.

返回值

和输入相同形状的张量

fit方法

对样本集进行处理，当‘featurewise_center’、’featurewise_std_normalization’、’zca_whitening’三者至少有一个设置为True时才能使用

参数

  • x 样本集，秩应该为 4。对于灰度图，通道轴的值应该为1
  • augment 是否使用随机样本扩张
  • rounds 随机扩张的次数
  • seed 随机数种子

返回值

无

flow方法

获取数据和标签，生成一批扩张的数据

参数

  • x 数据集，秩为4
  • y 标签
  • batch_size 一批的数据量
  • shuffle 是否将数据随机重排
  • sample_weight 样本权重
  • seed 随机数种子
  • save_to_dir 指定要保存的正在生成的增强图片的目录
  • save_prefix 保存图片的前缀
  • save_format 保存的图片的类型
  • subset ’training’或者’training’，只有在构造函数中设置了’validation_split’参数

返回值

一个(x, y)元祖的Iterator

flow_from_directory方法

参数

  • directory 目录路径
  • target_size 二维元祖，目标形状
  • color_mode ‘grayscale’或者’rgb’
  • classes 列表，如果没有，那么是第二级目录的目录名称
  • class_mode 'categorical', 'binary', 'sparse'，'input',或者是None。返回的标签数组的类型
  • batch_size 一批数据的大小
  • shuffle 是否将数据随机重排
  • seed 随机数种子
  • save_to_dir 指定要保存的正在生成的增强图片的目录
  • save_prefix 保存图片的前缀
  • save_format 保存的图片的类型
  • subset ’training’或者’training’，只有在构造函数中设置了’validation_split’参数
  • interpolation 当target_size和愿图片大小不同时进行差值的方式。有'nearest'、'bilinear'、'bicubic'

返回值

一个生成 (x, y) 元组的 DirectoryIterator,其中x是图像数组，y是标签

为转换生成随机参数

参数

  • seed 随机数种子

转换后的图片

standardize方法

参数

  • 需要标准化的一批输入

标准化后的数据

本文介绍tensorboard工具的使用

tensorboard 通常用于模型和参数的可视化，学习tensorboard对于深度神经网络可以用更好的理解

本文基于tensorboard官网的样例和代码

遇到的问题小结（MAC OS)

使用anaconda安装一些库时会安装最新版本，可能会和系统有些冲突的存在

protobuf

默认安装版本是3.11.2，需要降到3.8.0

如何使用tensorboard

在model的fit()方法传递callbacks参数

查看fit()方法的api

一些重要的参数

  • batch_size 一次训练的样本数，增大则训练速度变快，但内存消耗也随之增加
  • epochs 全部训练样本被训练的次数
  • validation_split 0-1之间的小数，表示划分多少训练样本为验证集
  • validation_data 验证样本，会覆盖validation_split，对于损失函数和精确度进行计算，但不会用于训练中
  • shuffle 是否对于训练样本进行随机排列
  • callbacks keras.callbacks.Callback类的对象，可以是tf.keras.callbacks.TensorBoard对象

定义一个tensorboard

参数

  • log_dir 存储tensorboard日志文件的路径
  • histogram_freq 计算激活值和权重直方图的频率
  • write_graph 是否查看计算图
  • update_freq 更新频率，‘batch’，‘epoch’或者是整数，即整数个batch

tf.keras.activations.rule

rule函数表达式如下
$$
f(x) = max(0,x)
$$
,而leaky rule表达式为
$$
f(x) = max(0.1x,x)
$$
在tf.keras中，对于rule有了更细致的定义，查看关于rule的代码如下

翻译一下就是
$$
f(x) = \begin{cases}
max\underline{}value & \text{if} x >= max\underline{}value\\
x & \text{ if } x < max\underline{}alue\ &\ x>=threshold \\
\alpha*x & \text{ if } x <max\underline{}value\ &\ x <threshold
\end{cases}
$$
默认情况下就是普通的rule函数，即f(x)=max(0,x)

参数

  • x 输入的张量
  • alpha 左半部分的斜率
  • max_value 整个函数能产生的最大值
  • threshold 两个部分的分界线

返回值

  • rule 激活函数

tf.keras.activations.selu

selu函数表达式如下
$$
f(x) = \begin{cases}
scale*x & \text{ if } x>=0 \\
scale *\alpha * (e^x -1)& \text{ if } x<0
\end{cases}
$$
其中$scale$和$\alpha$已经是确定的

  • $scale$ : 1.05070098
  • $\alpha$ : 1.67326324

  • x 输入的张量

返回值

  • selu激活函数

selu的目的主要是调整$x>=0$时函数的斜率
